# Apolingo
# ![Apolingo](APOLINGO-1.png)

Apolingo — учебный веб-проект для изучения английского языка в формате онлайн-обучения.

Проект разработан в рамках технологической практики и представляет собой информационную систему, позволяющую пользователям проходить регистрацию, авторизацию, выполнять учебные задания и сохранять результаты обучения.

## Основные возможности

* Регистрация пользователей
* Авторизация пользователей
* Прохождение учебных модулей
* Выполнение тестовых заданий
* Автоматическая проверка ответов
* Сохранение пользовательского прогресса
* Просмотр статистики обучения

## Используемые технологии

### Frontend

* HTML5
* CSS3
* JavaScript

### База данных

* Supabase
* PostgreSQL

## Структура базы данных

Проект использует следующие таблицы:

* users — пользователи системы
* lessons — учебные модули
* questions — вопросы
* answers — варианты ответов
* user_progress — результаты прохождения обучения
